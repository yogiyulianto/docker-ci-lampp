HTML Files and Structure

Html Structure

  1. This is a responsive layout
  2. <DIV>tag is used as container
  3. Each section is commented at just top and bottom of it
  4. Comments in HTML file exactly match with comments in CSS file
  5. Use recent version of any browser for better result.
  6. Tested in IE 11, Firefox, Chrome , Safari and Opera

To change the Fav Icon,Pick any variation of html template and open with any editor. Top of the page, short cut icon link has been given for fav icon. Here you change the fav icon name and place corresponding icon in the "images" folder. As well as, do the update of web site title and meta description for SEO point of view.

CSS Files and Structure

Boostrap framework and Custom style [style.css] have been used and corresponding style-sheets are available under "css" folder.

To change Header background image, Open the style sheet "style.css" and update the class "header1-bgimage,header2-bgimage ..." background url.

JavaScript

Boostrap framework, topmenu animation and smooth scroll has been used and corresponding javascripts are available under "js" folder.

Mailchimp Newsletter Subscription

IMPORTANT

Update your mailchimp embedded code for Mailchimp Newsletter Subscription in the HTML Page.

Update your mailchimp newsletter Error Message in the javascript file which is resides in js\vendor\subscribe\subscribe_validate.js

Working PHP Health Generation Form Email Integration

IMPORTANT

Update your contact receiver email id, name and subject for Contact Form inquery in the PHP file under mail/mail.php
